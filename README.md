# BodaKitaa Frontend

A premium React frontend for BodaKitaa, built with Vite, Tailwind CSS, GSAP, and i18next.

## Features
- **Premium UI**: Modern dark-themed design with Glassmorphism.
- **GSAP Animations**: Smooth, high-performance animations.
- **Multilingual Support**: English and Swahili support out of the box.
- **Responsive**: Mobile-first design using Tailwind CSS.
- **Security Focused**: Role-based access considerations in the architecture.

## Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Tech Stack
- **Framework**: React 18
- **Styling**: Tailwind CSS
- **Animations**: GSAP
- **Icons**: Lucide React
- **Internationalization**: i18next
- **Build Tool**: Vite
